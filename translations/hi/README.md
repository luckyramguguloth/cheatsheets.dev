## ✨ यह रिपोजिटरी क्यों?

- 🔌 **100% ऑफ़लाइन कार्य करता है** — एक बार क्लोन करें, हमेशा खोजें। कभी भी इंटरनेट की आवश्यकता नहीं।
- 🚫 **कोई विज्ञापन नहीं, कोई ट्रैकिंग नहीं, कोई पेवॉल नहीं** — शुद्ध जानकारी, कोई फालतू कचरा नहीं।
- 🌍 **सभी के लिए उपयुक्त** — डेवलपर्स, डेवऑप्स इंजीनियरों, गेमर्स, हार्डवेयर बिल्डरों, सिसाडमिन्स और छात्रों के लिए।

## ⚡ त्वरित शुरुआत

```bash
# रिपोजिटरी क्लोन करें
git clone https://github.com/yourusername/cheatsheets.dev
cd cheatsheets.dev

# खोज स्क्रिप्ट को निष्पादन योग्य बनाएं
chmod +x search.sh

# किसी भी विषय को तुरंत खोजें
./search.sh "docker stop"
```

## 🖥️ इंटरएक्टिव वेब ऐप (लोकलहोस्ट डॉक्स)

प्रीमियम ग्राफिकल अनुभव और तेजी से समझने के लिए, अंतर्निहित स्थिर वेब ऐप देखें:

```bash
# रिपॉजिटरी रूट में एक साधारण स्थानीय वेब सर्वर शुरू करें
python -m http.server 8000
```

फिर अपना ब्राउज़र खोलें और यहां जाएं:
👉 **`http://localhost:8000/docs/`**

**लोकलहोस्ट व्यू की विशेषताएं:**
- 🔍 **त्वरित खोज:** विज़ुअल हाइलाइट्स (`<mark>`) के साथ वास्तविक समय फ़ज़ी खोज।
- 🗂️ **इंटरएक्टिव टैब फ़िल्टर:** श्रेणी (Dev, DevOps, Gaming, Hardware) द्वारा गतिशील रूप से चीटशीट्स फ़िल्टर करें।
- 🔮 **प्रीमियम रीडर मोडल:** शून्य पेज रीलोड के साथ एक सुंदर ग्लासमोर्फिक मोडल में मार्कडाउन शीट प्रस्तुत करता है।
- 📋 **क्लिपबोर्ड पर कॉपी करें:** सभी कोड ब्लॉक पर एक-क्लिक कॉपी बटन।
- 📴 **100% ऑफ़लाइन-सुरक्षित:** बिना किसी इंटरनेट कनेक्शन के पूरी तरह से काम करता है।

## 📴 ऑफ़लाइन उपयोग कैसे करें

एक बार रिपॉजिटरी क्लोन करें और काम हो गया — हर चीटशीट एक प्लेन मार्कडाउन फ़ाइल है जो किसी भी संपादक, टर्मिनल या मार्कडाउन व्यूअर में सुंदर दिखती है। फ़ाइलों को सीधे खोलें, `./search.sh` का उपयोग करें, या [Obsidian](https://obsidian.md/), [Typora](https://typora.io/), या VS Code के साथ ब्राउज़ करें।
